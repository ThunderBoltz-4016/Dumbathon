# Dumbathon

## Accessing this repository

You can clone this repository locally using Git:

```bash
git clone https://github.com/ThunderBoltz-4016/Dumbathon.git
```

After cloning, change into the folder:

```bash
cd Dumbathon
```

If you already have the repo locally and want to fetch the latest changes:

```bash
git pull origin main
```

## Notes

- The project is a Next.js/Tailwind app located under `zombie-black-market/`.
- Run `npm install` inside `zombie-black-market/` before running the dev server.

🛡️ CAPTCHA "Cheat Sheet"
The CAPTCHAs are designed to be frustrating. Here is how to pass each type:

Traffic Lights / Carbon Atoms: Select nothing on the grid and just click "Verify".
Prove you are NOT a robot: Click any of the options (they are all self-incriminating, e.g., "I AM a robot").
Invisible Verify Button: Move your mouse around until you find a clickable area that doesn't look like a button.
No Mouse Movement: Hands off! Do not move your cursor for 15 seconds straight.
Click Count: Click the button exactly 50 times. If you click 51, it resets.
Profound Longing: Select the 1st, 3rd, 5th, 7th, and 9th images (Indices 0, 2, 4, 6, 8).
Click all Bridges: Select the 2nd, 5th, and 8th images (Indices 1, 4, 7). The others are dental bridges (teeth).
Trolley Problem: Click any of the moral dilemmas to proceed.
